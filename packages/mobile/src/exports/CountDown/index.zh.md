# CountDown 倒计时 <ImportCost name="CountDown" />

倒计时，逐秒递减。

## 属性说明

| 属性        | 说明                                      | 类型   | 默认值 | 提供的值 |
| :---------- | :---------------------------------------- | :----- | :----- | :------- |
| deadline    | 最后截至的事件，应为时间戳(单位为毫秒 ms) | Number | -      |
| localOffset | 局部偏置                                  | Number | -      |
