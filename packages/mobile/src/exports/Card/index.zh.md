# Card 卡片 <ImportCost name="Card" />

通用卡片容器。

## 属性说明

| 属性        | 说明                       | 类型          | 默认值 | 提供的值 |
| :---------- | :------------------------- | :------------ | :----- | :------- |
| title       | 卡片的主标题               | string \| JSX | -      | -        |
| titleExtra  | 卡片右上角的操作区域       | string \| JSX | -      | -        |
| footer      | 卡片的底部                 | string \| JSX | -      | -        |
| footerExtra | 卡片右下角的操作区域       | string \| JSX | -      | -        |
| after       | 卡片之外下半部分衔接的区域 | string \| JSX | -      | -        |
