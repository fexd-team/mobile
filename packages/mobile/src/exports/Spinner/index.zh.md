## 属性说明

| 属性  | 说明                                       | 类型   | 默认值 | 提供的值 |
| :---- | :----------------------------------------- | :----- | :----- | :------- |
| delay | 延迟出现时间（防止闪烁），单位为毫秒（ms） | number | 0      | -        |
