# Breadcrumb 面包屑 <ImportCost name="Breadcrumb" />

显示当前页面在系统层级结构中的位置，并能够点击。

## 属性说明

| 属性 | 说明                               | 类型         | 默认值 | 提供的值 |
| :--- | :--------------------------------- | :----------- | :----- | :------- |
| data | 必填，当前页面的位置数据，需按顺序 | ReactChild[] | -      | -        |

## 事件属性说明

| 事件名称 | 说明                     | 回调参数         |
| :------- | :----------------------- | :--------------- |
| onClick  | 绑定点击某一项触发的事件 | 点击的元素的数据 |
