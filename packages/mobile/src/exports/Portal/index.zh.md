# Portal 渲染视窗 <ImportCost name="Portal" />

Portal 将提供一种将子节点渲染到 DOM 节点中的方式，该节点存在于 DOM 组件的层次结构之外。

## 属性说明

| 属性 | 说明           | 类型 | 默认值        | 提供的值 |
| :--- | :------------- | :--- | :------------ | :------- |
| to   | 将加入的父节点 | dom  | document.body | -        |
